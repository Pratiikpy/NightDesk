# NightDesk Alpha Gateway — Submission Copy

## 200-word description

NightDesk Alpha Gateway is the autonomous fair-value, safety, and trust layer for Bitget tokenized US
stocks. Tokenized stocks trade 24/7, but the real US stock market only prices during NYSE hours, so
off-hours rTokens drift away from fair value while the stock perp — a blended issuer index — hides the
true gap. NightDesk anchors each token to the real NYSE price, exposes the dislocation the perp hides,
explains whether each gap is noise or real news, and requires every proposed trade to pass a
certificate firewall before execution. An autonomous Alpha Factory also searches rToken/perp/equity-gap
strategies and freezes a champion, with PnL reported honestly as in-sample execution evidence — not a
profit promise.

The system runs a full autonomous loop: Bitget market data plus equity anchors feed PegWatch;
perception reads macro/news context; the Alpha Factory searches candidates; a Qwen-powered
multi-role council decides trade or no-trade; 15 deterministic gates enforce risk; BitSim simulates
realistic fills; and every trade, abstention, blocked intent, strategy trial, and champion decision
is exported as judge evidence. NightDesk also exposes MCP tools, an HTTP firewall, and a TypeScript
SDK so external agents can call `evaluate_intent` before placing tokenized-stock orders.

This is Trading Infrastructure and Stock AI Trading: an alpha research lab, certification layer,
agent firewall, benchmark arena, signed audit ledger, and runnable evidence pack for safer and more
profitable Bitget AI agents.

## 3-minute demo story

1. **Problem:** Tokenized US stocks trade all night, but the true stock anchor is closed. Agents can
   trade blind, and the perp can hide the real rToken gap.
2. **Truth gap:** Show the cockpit’s Bitget Native Proof and Safety Gateway panels: read-only
   Bitget data, external MCP proof, paper log, and ledger proof all show PASS.
3. **Alpha Factory:** Show `evidence/alpha-factory/manifest.json`: candidates, trials, rejected
   configs, frozen champion, expected-vs-actual log. This proves the agent searches for profit
   instead of hand-picking one strategy.
4. **Judge Cockpit:** Open `evidence/judge-cockpit/index.html`: one screen shows Alpha Factory,
   raw-PnL championship, two-champion Championship Mode, safety gateway, doctor, and data-health
   status.
5. **Risk desk:** Show `npm run dashboard`. Point to true gap vs NYSE anchor, perp gap, cause, and
   action.
6. **Gateway:** Show `evidence/sample-inputs/safe-buy-intent.json`, then
   `evidence/sample-outputs/safe-buy-verdict.json`. NightDesk returns ALLOW/ALLOW_CAPPED/REJECT.
7. **Unsafe path:** Show `unsafe-sell-verdict.json`: a rich/short or policy-violating intent is
   rejected before execution.
8. **Paper record:** Show `evidence/alpha-factory/live-paper-trading-log.csv` and
   `evidence/trading-log/guarded-replay/guarded-replay-paper-trading-log.csv`:
   timestamp, asset, direction, price, quantity, balance change, certificate, verdict, policy,
   reason, ledger hash. The raw-PnL layer includes a PnL Champion for the green-number case and a
   Safety Champion for the production case; both are frozen separately and exported with paper logs.
9. **Overfit proof:** Show `evidence/championship/champion-overfit-card.md`: it labels same-sample
   PnL honestly and reports stress checks instead of hiding fragility.
10. **OOS proof:** Show `evidence/oos/session-bank/session-quality-report.md`: current sessions are
   signed and future sessions append through the running daemon.
11. **Shadow proof:** Show `evidence/shadow-gateway/actual-vs-guarded.csv`: it reports missed profit
   and blocked loss instead of hiding uncomfortable counterfactuals.
12. **Fast paper proof:** Show `evidence/trading-log/nightdesk-paper-trading-log.csv`: timestamp, asset,
   direction, price, quantity, balance change, certificate, verdict, policy, reason, ledger hash.
13. **Proof:** Show `npm run judge:max`: tests pass, evidence artifacts verify, signed ledger exists,
   paper logs exist, alpha artifacts exist, and the max evidence manifest verifies. Mention that
   `npm run judge:max:full` regenerates the full stack including Alpha Championship and Alpha
   Factory.
14. **Integration:** Show `sdk/examples/external-agent.ts` and MCP `evaluate_intent`. The product is
   not only an agent; it is the gateway other agents pass through.
